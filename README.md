# Dashboard-anomalies-industrielles
Dashboard de détection d’anomalies sur la consommation électrique horaire d’un site industriel / réseau local, permettant d’identifier automatiquement des comportements anormaux (surconsommation, arrêts...). Les anomalies sont détectées via un modèle non supervisé (IsolationForest / z-score) et visualisées sur un dashboard Streamlit interactif.
